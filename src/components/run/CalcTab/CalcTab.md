# CalcTab

The "Calc" tab of a run page. A damage calculator, currently showing the
attacker's `PokemonPanel` selected from the run's box.

## Props

| Prop   | Type   | Required | Default | Description                                        |
| ------ | ------ | -------- | ------- | -------------------------------------------------- |
| `game` | `Game` | Yes      | -       | The current game, passed through to `PokemonPanel` |
| `run`  | `Run`  | Yes      | -       | The current run, passed through to `PokemonPanel`  |
