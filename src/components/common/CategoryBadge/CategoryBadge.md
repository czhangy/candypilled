# CategoryBadge

A small badge image for a move's damage category (e.g.
`/move_categories/physical.png`).

## Props

| Prop       | Type     | Required | Default | Description                        |
| ---------- | -------- | -------- | ------- | ---------------------------------- |
| `category` | `string` | Yes      | -       | The category whose badge is shown  |
| `width`    | `number` | Yes      | -       | The badge's rendered width, in px  |
| `height`   | `number` | Yes      | -       | The badge's rendered height, in px |
